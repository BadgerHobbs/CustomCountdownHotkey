# CustomCountdownHotkey

Simple application for setting up a variety of timers which run on the press of a specified hotkey regardless of whether the application is set as the active window.

## Example Config
```json
{
  "HotKeys": [
    {
      "HotKey": "0",
      "StartTimeSeconds": "30"
    },
    {
      "HotKey": "9",
      "StartTimeSeconds": "5"
    }
  ]
}
```
